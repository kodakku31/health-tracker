# 健康管理アプリ UI要件定義

## アプリケーション概要
Next.jsとSupabaseを使用した現代的な健康管理アプリケーション。ユーザーの健康データを簡単に記録・管理し、健康状態の可視化と改善を支援します。

## ページ構造
```
├── / (ホームページ)
│   ├── /auth
│   │   ├── /login (ログインページ)
│   │   └── /signup (新規登録・プロフィール設定)
│   │
│   ├── /dashboard (ダッシュボード)
│   │   ├── 健康スコアの表示
│   │   │   ├── 総合スコア
│   │   │   ├── 運動スコア
│   │   │   ├── 睡眠スコア
│   │   │   ├── 栄養スコア
│   │   │   └── バイタルスコア
│   │   ├── 最近の記録サマリー
│   │   └── クイックアクション
│   │
│   ├── /health-records (健康記録)
│   │   ├── /vital-signs (バイタルサイン記録)
│   │   │   ├── 体重記録
│   │   │   ├── 血圧記録
│   │   │   ├── 体温記録
│   │   │   └── 心拍数記録
│   │   ├── /exercise (運動記録)
│   │   │   ├── 運動タイプ選択
│   │   │   ├── 運動強度設定
│   │   │   ├── 時間・距離記録
│   │   │   └── カロリー消費表示
│   │   └── /sleep (睡眠記録)
│   │       ├── 睡眠時間記録
│   │       ├── 睡眠品質評価
│   │       └── 睡眠ステージ記録

│   │   │   ├── 写真アップロード
│   │   │   └── 栄養素表示
```

## UI要件詳細

### 1. 共通要素
- モダンでクリーンなデザイン
- レスポンシブ対応（モバイル、タブレット、デスクトップ）
- ダークモード対応
- ローディング状態の表示
- エラー処理とフィードバック
- データ入力バリデーション
- 単位切り替え（metric/imperial）対応

### 2. ユーザー認証・プロフィール (/auth)
#### ログイン画面
- メールアドレス/パスワードログイン
- ソーシャルログイン（Google, Apple）
- パスワードリセット機能

#### プロフィール設定画面
- 基本情報入力
  - 表示名
  - 生年月日
  - 性別
  - 身長
  - 目標体重
  - 活動レベル
- プロフィール更新履歴表示

### 3. ダッシュボード (/dashboard)
- 健康スコアの大きな表示
  - 総合スコア（0-100）
  - コンポーネントスコア（運動、睡眠、栄養、バイタル）
  - インサイト表示
- データ可視化（グラフ、チャート）
- クイックアクション
  - バイタルサイン記録
  - 運動記録
  - 食事記録
  - 睡眠記録
- 最近の記録サマリー
- 目標達成状況

### 4. バイタルサイン記録 (/health-records/vital-signs)
- 記録フォーム
  - 体重入力（0-500 kg）
  - 血圧入力（収縮期: 0-300 mmHg, 拡張期: 0-200 mmHg）
  - 体温入力（30-45 ℃）
  - 心拍数入力（bpm）
  - メモ入力
- トレンドグラフ表示
- 履歴表示・編集

### 5. 運動記録 (/health-records/exercise)
- 運動タイプ選択
  - ウォーキング
  - ランニング
  - サイクリング
  - 水泳
  - ウェイトトレーニング
  - ヨガ
  - その他
- 運動強度設定（低・中・高）
- 時間設定（開始・終了）
- 距離・歩数入力
- カロリー消費表示
- メモ機能

### 6. 睡眠記録 (/health-records/sleep)
- 睡眠時間入力（開始・終了）
- 睡眠品質評価（poor・fair・good・excellent）
- 睡眠ステージ記録
  - 深い睡眠
  - 浅い睡眠
  - レム睡眠
- 中断回数記録
- メモ機能
- 睡眠分析グラフ

### 7. 食事記録 (/nutrition/meal-log)
- 食事タイプ選択（朝食・昼食・夕食・間食）
- 食品検索・登録システム
  - 食品名
  - 量
  - 単位（g・ml・servings）
  - カロリー
  - 栄養素（タンパク質・炭水化物・脂質・食物繊維・糖質）
- 写真アップロード機能
- 総カロリー表示
- メモ機能

### 8. 目標設定 (/goals)
- 目標タイプ選択
  - 体重
  - 運動
  - 睡眠
  - 栄養
  - 水分摂取
  - 歩数
- 目標値設定
- 期間設定（開始日・終了日）
- 進捗表示
- リマインダー設定（日次・週次・月次）
- メモ機能

### 9. 設定 (/settings)
- テーマ設定
  - ライトモード
  - ダークモード
  - システム設定に従う
- 通知設定
  - 目標リマインダー
  - 測定リマインダー
  - インサイト通知
  - その他リマインダー
- 単位設定（metric/imperial）
- 言語設定
- プライバシー設定
  - データ共有設定
  - 匿名分析の許可
